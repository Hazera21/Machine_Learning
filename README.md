# Machine_Learning

Regressor.ipynb - Linear Regression Model to predict Expenses based on input Features. Built the model using AWS Sagemaker, Sikit Learn Libraries.

Image_Classification.py - Deep Learning model, CNN is used to train the model with food images and tested with test data.
Deployed using Streamlit for intercative inferences.

Sentiment_Analysis.py - Machine learning model built using Naive Baye's algorithm to predict the sentiment analysis. Trained using IMDB dataset.
